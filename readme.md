# Linux setup

## Checkout this repository
```bash
ssh-keygen -t ed25519 -C "<email>"
git config --global user.email <>
git config --global user.name <>
git clone git@github.com:ikempf/.dotfiles-linux.git ~/dotfiles
```

## Install dotfiles with stow
```bash
# Install stow first
brew install stow
cd ~/dotfiles
stow */
