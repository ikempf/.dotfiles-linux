# Keymaps

Useful keymaps worth keeping around as I learn or reuse them.

- `<leader>sk` - keymaps
- `<C-h>` / `<C-j>` / `<C-k>` / `<C-l>` - move between Neovim and tmux panes
- `<leader>al` - list attempts
- `<leader>an` - new attempt
- `w` / `b` / `e` - move by word
- `0` / `^` / `$` - start of line / first non-blank / end of line
- `gg` / `G` - top / bottom of file
- `%` - jump to matching bracket or parenthesis
- `*` / `#` - search for word under cursor forward / backward
- `n` / `N` - next / previous search match
- `ciw` - change inner word
- `diw` - delete inner word
- `yiw` - yank inner word
